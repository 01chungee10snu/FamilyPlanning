비교 연구: 서로 다른 길

인터랙티브 맵
방글라데시 - 성공 모델
• TFR: 6.3 → 3.0 (53% 감소)
• CPR: 8% → 58% (7배 증가)
• 여성 현장요원 5만명
파키스탄 - 제한적 성과
• TFR: 6.3 → 4.3 (32% 감소)
• CPR: 5% → 28% (5.6배 증가)
• 시설 중심 접근

🏆 방글라데시 성공 요인
